1
Create a Google Sheet
Open sheets.google.com → click + Blank → name it TaskFlow Pro.
Keep this tab open — you'll need it in the next step.
💡 Your data will be stored in this sheet. You can view, filter and even edit data directly in Google Sheets anytime.
2
Open Apps Script
In your Google Sheet, click Extensions in the top menu → click Apps Script. A new tab will open with a code editor.

4
Deploy as Web App
1. Click Deploy (top right) → New deployment
2. Click the ⚙️ gear next to "Select type" → choose Web app
3. Set Execute as: Me  |  Who has access: Anyone
4. Click Deploy → click Authorize access → sign in with Google
5. Copy the Web app URL that appears
⚠️ Make sure "Who has access" is set to Anyone — otherwise the app can't connect.
